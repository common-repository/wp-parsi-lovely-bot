== Installation ==

1. Upload `wp-parsi-lovely-bot` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Replace `the_content()` Function In Theme File (index, single, search, page, tag, archive, ...) with `the_content_bot_wpp("more content ...")` .
